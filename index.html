Welcome to the AntiSQLi GitHub Page.

The AntiSQLi library is a free and easy to use library for reducing risk from SQL injection (SQLi) attacks, the current #1 Internet attack today. It auto-parameterizes untrusted data for you and best of all uses coding patterns and classes that you're already familiar with.


SqlParameterizedQuery pQuery = new SqlParameterizedQuery();
pQuery.LoadQuery("SELECT * FROM Users WHERE UserName = {0} AND Email = {1}",UntrustedDataUserName, UntrustedDataEmail);
...
pQuery.Connection = new SqlConnection("");
pQuery.Command.ExecuteNonQuery();
